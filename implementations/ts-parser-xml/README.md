# ts-parser-xml

Simple XML parser implementation

- Project template: [[GitHub] erkobridee/vitest-ts](https://github.com/erkobridee/vitest-ts)

## Commands

### Setup

```
npm i
```

### Single execution

> to check the output at the terminal

#### Lexer

```
npm run lexer
```

#### Parser

```
npm run parser
```

### Development

The following command will start the vitest

```
npm start

or

npm run dev
```
